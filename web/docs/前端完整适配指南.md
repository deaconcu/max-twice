## 通用更新模式

### 1. 添加imports
```javascript
import { APPROVAL_STATE, PROGRESS_STATE, APPROVAL_STATE_TEXT, PROGRESS_STATE_TEXT } from '@/constants/statusConstants';
```

### 2. 暴露常量给模板
```javascript
const { APPROVAL_STATE, PROGRESS_STATE } = { APPROVAL_STATE, PROGRESS_STATE };
```

### 3. 更新状态比较
```javascript
// 审批状态
'SUBMITTED' → APPROVAL_STATE.SUBMITTED (0)
'APPROVED' → APPROVAL_STATE.APPROVED (1)  
'REJECTED' → APPROVAL_STATE.REJECTED (2)

// 进度状态 (注意字段名变化)
item.status === 'NOT_STARTED' → item.state === PROGRESS_STATE.NOT_STARTED (0)
item.status === 'IN_PROGRESS' → item.state === PROGRESS_STATE.IN_PROGRESS (1)
item.status === 'COMPLETED' → item.state === PROGRESS_STATE.COMPLETED (2)
```

### 4. 更新状态设置
```javascript
// 审批状态
item.state = APPROVAL_STATE.APPROVED;

// 进度状态  
item.state = PROGRESS_STATE.COMPLETED;
```

### 5. 更新文本显示
```javascript
// 审批状态文本
APPROVAL_STATE_TEXT[item.state]

// 进度状态文本
PROGRESS_STATE_TEXT[item.state]
```

## 重要注意事项

### 🚨 字段名变更
- UserCourse和UserRoadmap的字段名从 `status` 改为 `state`
- 前端代码中所有 `item.status` 都要改为 `item.state`

### 🚨 API调用
- 确保API调用传递整型值 (0,1,2) 而非字符串
- learnService中相关方法的参数要使用常量

### 🚨 模板更新
- 所有v-if条件中的状态比较要使用常量
- 例：`v-if="item.state === PROGRESS_STATE.COMPLETED"`
