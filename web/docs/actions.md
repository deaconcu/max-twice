本页描述网站所有页面和用户行为，用来帮助测试

# auth

- 输入域名，首次进入时，跳转登陆注册页面
- 输入邮箱/密码登陆
- 输入邮箱/密码注册
- 输入验证码校验邮箱

# 职业

- 点击左侧职业标签，加载列表页
- 选择一级分类，显示 1. 二级分类 2. 职业列表
- 选择二级分类，显示 职业列表
- 列表滚动加载
- 创建职业
  - 输入职业名称/描述/分类 申请创建职业
  - 职业在管理后台审核通过/不通过时，收到系统消息

# 课程

- 点击左侧课程标签，加载列表页
- 选择一级分类，显示 1. 二级分类 2. 课程列表
- 选择二级分类，显示 课程列表
- 列表滚动加载
- 创建课程
  - 输入课程名称/描述/分类 申请创建课程
  - 课程在管理后台审核通过/不通过时，收到消息

# 职业详情

- 点击职业进入职业详情页
- 职业路径滚动加载

# 创建职业路径

- 搜索课程，显示搜索结果，按相似度排序（完全匹配排第一，前缀大于后缀，越短越排在前面，默认只显示50条）
- 编辑器的各项功能
  - 添加节点
  - 建立连接
  - 删除节点
  - 删除连接
  - 一次删除多个节点和连接
  - 重置画布
  - 自动布局
  - 保存草稿
  - 保存并发布
- 保存并发布以后，检查消息是否发送

# 课程详情

- 点击课程，进入课程详情页
  - 检查信息是否完整
  - 检查子课程是否加载
- 收藏课程
- 收藏子课程

# 阅读课程

- 点击课程详情页的阅读课程进入页面
