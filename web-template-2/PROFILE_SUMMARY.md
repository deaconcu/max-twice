## 📊 功能特性

### 1. 个人信息 Tab
- 可编辑字段：用户名、邮箱、个人简介
- 加入日期（只读）
- 保存/取消按钮
- 成功提示（3秒后自动消失）

### 2. 学习进度 Tab
- 环形进度图（总体进度 45%）
- 4个统计指标卡片
- 4门正在学习的课程
- 每门课程显示：图标、进度条、完成节点数
- 点击课程跳转到课程详情

### 3. 数据统计 Tab
- 4个统计卡片（学习课程、完成课程、学习天数、学习时长）
- 增长趋势指示（向上/向下箭头 + 百分比）
- 时间段切换（今天、7天、30天、全部）
- 简单柱状图展示学习趋势
- 动态数据根据时间段变化

### 4. 订阅管理 Tab
- 6门订阅课程展示
- 课程信息：图标、名称、描述、学习人数、分类
- 取消关注按钮（带确认对话框）
- 点击卡片跳转到课程详情
- 底部显示订阅总数

### 5. 关注的人 Tab
- 6个用户卡片展示
- 用户信息：头像（首字母）、姓名、简介、文章数、粉丝数
- 取消关注按钮
- 顶部显示关注数和粉丝数统计

### 6. 创建的目录 Tab
- 3个目录卡片
- 目录信息：名称、描述、内容数量、浏览量、点赞数
- 公开/私密状态标签
- 创建日期

### 7. 创建的文章 Tab
- 3篇文章列表（水平布局）
- 文章信息：封面图、标题、摘要、标签
- 统计数据：浏览量、点赞数、评论数
- 发布/草稿状态标签
- 编辑和删除按钮

### 8. 记忆卡片组 Tab
- 4个卡片组展示
- 卡片组信息：图标、名称、描述、创建日期
- 统计数据：总卡片数、已复习数、待复习数
- 进度条显示复习进度
- 最近复习时间
- 状态标签（进行中/已归档）
- 编辑和删除功能（带确认对话框）
- 底部总计统计

### 9. 创建的路线图 Tab
- 3个路线图卡片
- 路线图信息：名称、职业、描述、节点数
- 使用统计：使用人数、收藏数
- 状态标签（公开/私密/草稿）
- 缩略图占位符
- 查看、编辑、删除按钮

## 🔧 技术实现

### 路由集成
```typescript
// 支持 URL 查询参数
/profile?tab=info      // 个人信息
/profile?tab=studying  // 正在学习
/profile?tab=stats     // 数据统计
/profile?tab=courses   // 关注的课程
// ... 等等
```

### 组件通信
```typescript
// 父传子
<UserInfoTab :user-info="userInfo" />

// 子传父
@update="handleUpdateUserInfo"
```

### Mock 数据策略
- 所有数据在组件内部定义
- 使用 `ref()` 创建响应式数据
- 模拟真实的数据结构和统计
- 支持本地操作（取消关注、删除等）

## 📝 使用说明

### 访问方式
```
http://localhost:5173/profile
```

### Tab 切换
1. 点击水平 Tab 导航栏切换
2. URL 会自动更新为 `/profile?tab=xxx`
3. 刷新页面会保持当前 Tab
4. 可以直接访问指定 Tab，例如 `/profile?tab=studying`

### 功能测试
1. **个人信息**：编辑字段 → 点击保存 → 查看成功提示
2. **学习进度**：点击课程卡片 → 跳转到课程详情
3. **数据统计**：切换时间段 → 查看数据和图表变化
4. **订阅管理**：点击心形图标 → 确认取消 → 课程消失
5. **关注的人**：点击取消关注 → 用户移除
6. **记忆卡片组**：点击删除 → 确认 → 卡片组消失
