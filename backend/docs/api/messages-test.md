# 消息管理接口测试文档

## 测试概览

本文档描述消息管理模块的测试用例，包括功能测试、边界测试、异常测试等。

**测试范围**:
- 按分类获取消息接口
- 邀请用户接口
- 消息分页功能
- 消息类型过滤
- 权限验证
- 参数校验

## 1. 按分类获取消息接口测试

### 1.1 功能测试

#### 测试用例 1.1.1: 获取互动消息列表
- **测试目的**: 验证能正确获取互动消息
- **前置条件**:
  - 用户已登录
  - 数据库中存在互动消息数据
- **测试步骤**:
  1. 使用有效token发起请求
  2. 设置参数 category=1
  3. 不传递 lastId 和 type 参数
- **预期结果**:
  - 返回状态码 200
  - data 数组包含互动消息
  - 消息按 id 倒序排列
  - 最多返回 20 条消息
  - 每条消息包含 id, type, createdAt 字段
  - 根据消息类型包含对应的扩展字段（follower/commenter/upvoter/inviter）
