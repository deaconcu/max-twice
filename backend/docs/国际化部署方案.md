# 国际化部署方案

## 概述

本文档描述了学习平台的多语言网站国际化部署方案。采用完全独立部署的架构，支持中文和英文两个版本的网站。

## 1. 架构设计

### 1.1 部署架构
```
zh.yoursite.com  →  中文版服务器/容器 (端口 8080)
en.yoursite.com  →  英文版服务器/容器 (端口 8081)
```

### 1.2 技术栈
- **后端**: Spring Boot + Maven
- **容器化**: Docker + Docker Compose
- **反向代理**: Nginx
- **数据库**: MySQL (独立数据库)
- **SSL**: Let's Encrypt

## 2. 项目结构

### 2.1 配置文件结构

### 3.2 国际化配置类

#### LocaleConfig.java
```java
@Configuration
public class LocaleConfig {
    
    @Value("${app.default-language:zh}")
    private String defaultLanguage;
    
    @Bean
    public LocaleResolver localeResolver() {
        FixedLocaleResolver resolver = new FixedLocaleResolver();
        Locale locale = "zh".equals(defaultLanguage) ? 
            Locale.SIMPLIFIED_CHINESE : Locale.ENGLISH;
        resolver.setDefaultLocale(locale);
        return resolver;
    }
    
    @Bean
    public MessageSource messageSource() {
        ResourceBundleMessageSource messageSource = new ResourceBundleMessageSource();
        messageSource.setBasename("messages/messages_" + defaultLanguage);
        messageSource.setDefaultEncoding("UTF-8");
        messageSource.setCacheSeconds(3600);
        messageSource.setUseCodeAsDefaultMessage(true);
        return messageSource;
    }
}
```

## 4. Docker部署

### 4.1 Dockerfile
```dockerfile
FROM openjdk:17-jre-slim

WORKDIR /app

COPY target/learn-api-*.jar app.jar

EXPOSE 8080

ENTRYPOINT ["java", "-jar", "app.jar"]
```

### 4.2 docker-compose.yml
```yaml
version: '3.8'

services:
  mysql-zh:
    image: mysql:8.0
    container_name: mysql-zh
    environment:
      MYSQL_ROOT_PASSWORD: ${MYSQL_ROOT_PASSWORD}
      MYSQL_DATABASE: learn_zh
    ports:
      - "3306:3306"
    volumes:
      - mysql_zh_data:/var/lib/mysql
    networks:
      - learn-network

  mysql-en:
    image: mysql:8.0
    container_name: mysql-en
    environment:
      MYSQL_ROOT_PASSWORD: ${MYSQL_ROOT_PASSWORD}
      MYSQL_DATABASE: learn_en
    ports:
      - "3307:3306"
    volumes:
      - mysql_en_data:/var/lib/mysql
    networks:
      - learn-network

  chinese-site:
    build: 
      context: .
      dockerfile: Dockerfile
    container_name: learn-zh
    environment:
      - SPRING_PROFILES_ACTIVE=zh
      - DB_USERNAME=${DB_USERNAME}
      - DB_PASSWORD=${DB_PASSWORD}
    ports:
      - "8080:8080"
    depends_on:
      - mysql-zh
    networks:
      - learn-network
    restart: unless-stopped

  english-site:
    build:
      context: .
      dockerfile: Dockerfile
    container_name: learn-en
    environment:
      - SPRING_PROFILES_ACTIVE=en
      - DB_USERNAME=${DB_USERNAME}
      - DB_PASSWORD=${DB_PASSWORD}
    ports:
      - "8081:8080"
    depends_on:
      - mysql-en
    networks:
      - learn-network
    restart: unless-stopped

volumes:
  mysql_zh_data:
  mysql_en_data:

networks:
  learn-network:
    driver: bridge
```

## 5. Nginx配置

### 5.1 /etc/nginx/sites-available/multilingual
```nginx
# 中文站点
server {
    listen 80;
    server_name zh.yoursite.com;
    
    # HTTP重定向到HTTPS
    return 301 https://$server_name$request_uri;
}

server {
    listen 443 ssl http2;
    server_name zh.yoursite.com;
    
    # SSL配置
    ssl_certificate /etc/letsencrypt/live/zh.yoursite.com/fullchain.pem;
    ssl_certificate_key /etc/letsencrypt/live/zh.yoursite.com/privkey.pem;
    ssl_protocols TLSv1.2 TLSv1.3;
    ssl_ciphers HIGH:!aNULL:!MD5;
    
    # 反向代理到中文版应用
    location / {
        proxy_pass http://localhost:8080;
        proxy_set_header Host $host;
        proxy_set_header X-Real-IP $remote_addr;
        proxy_set_header X-Forwarded-For $proxy_add_x_forwarded_for;
        proxy_set_header X-Forwarded-Proto $scheme;
        proxy_buffering off;
    }
    
    # 静态资源缓存
    location ~* \.(js|css|png|jpg|jpeg|gif|ico|svg)$ {
        proxy_pass http://localhost:8080;
        expires 1y;
        add_header Cache-Control "public, immutable";
    }
}
```

## 6. 数据库初始化

### 6.1 数据库创建脚本
```sql
-- 创建中文数据库
CREATE DATABASE IF NOT EXISTS learn_zh 
DEFAULT CHARACTER SET utf8mb4 
DEFAULT COLLATE utf8mb4_unicode_ci;

-- 创建英文数据库
CREATE DATABASE IF NOT EXISTS learn_en 
DEFAULT CHARACTER SET utf8mb4 
DEFAULT COLLATE utf8mb4_unicode_ci;

-- 创建用户（可选）
CREATE USER IF NOT EXISTS 'learn_user'@'%' IDENTIFIED BY 'your_password';
GRANT ALL PRIVILEGES ON learn_zh.* TO 'learn_user'@'%';
GRANT ALL PRIVILEGES ON learn_en.* TO 'learn_user'@'%';
FLUSH PRIVILEGES;
```

### 6.2 数据迁移脚本
```bash
#!/bin/bash
# migrate-databases.sh

echo "Migrating Chinese database..."
mysql -h localhost -P 3306 -u root -p learn_zh < sql/schema.sql
mysql -h localhost -P 3306 -u root -p learn_zh < sql/data_zh.sql

echo "Migrating English database..."
mysql -h localhost -P 3307 -u root -p learn_en < sql/schema.sql
mysql -h localhost -P 3307 -u root -p learn_en < sql/data_en.sql

echo "Database migration completed."
```

## 7. 部署脚本

### 7.1 启动脚本 (start.sh)
```bash
#!/bin/bash

echo "Starting multilingual learning platform..."

# 检查Docker是否运行
if ! docker info > /dev/null 2>&1; then
    echo "Error: Docker is not running"
    exit 1
fi

# 构建并启动服务
echo "Building and starting services..."
docker-compose up -d --build

# 等待服务启动
echo "Waiting for services to start..."
sleep 30

# 检查服务状态
echo "Checking service status..."
docker-compose ps

echo "Deployment completed!"
echo "Chinese site: https://zh.yoursite.com"
echo "English site: https://en.yoursite.com"
```

### 7.2 停止脚本 (stop.sh)
```bash
#!/bin/bash

echo "Stopping multilingual learning platform..."

docker-compose down

echo "All services stopped."
```

### 7.3 更新脚本 (update.sh)
```bash
#!/bin/bash

echo "Updating multilingual learning platform..."

# 拉取最新代码
git pull origin main

# 重新构建并启动
docker-compose down
docker-compose up -d --build

echo "Update completed!"
```

```bash
# 为两个子域名申请证书

# 设置自动续期
sudo crontab -e
# 添加以下行：
# 0 2 * * * /usr/bin/certbot renew --quiet
```

```yaml
            <fileNamePattern>${LOG_PATH}/application.%d{yyyy-MM-dd}.log</fileNamePattern>
            <maxHistory>30</maxHistory>
        </rollingPolicy>
        <encoder>
            <pattern>%d{HH:mm:ss.SSS} [%thread] %-5level %logger{36} - %msg%n</pattern>
        </encoder>
    </appender>
    
    <root level="INFO">
        <appender-ref ref="FILE"/>
    </root>
</configuration>
```

## 10. CI/CD配置

### 10.1 GitHub Actions
```yaml
# .github/workflows/deploy.yml
name: Deploy Multilingual Sites

on:
  push:
    branches: [ main ]

jobs:
  deploy:
    runs-on: ubuntu-latest
    
    steps:
    - name: Checkout code
      uses: actions/checkout@v3
    
    - name: Set up JDK 17
      uses: actions/setup-java@v3
      with:
        java-version: '17'
        distribution: 'temurin'
        
    - name: Cache Maven dependencies
      uses: actions/cache@v3
      with:
        path: ~/.m2
        key: ${{ runner.os }}-m2-${{ hashFiles('**/pom.xml') }}
        
    - name: Build application
      run: mvn clean package -DskipTests
      
    - name: Deploy to server
      uses: appleboy/ssh-action@v0.1.5
      with:
        host: ${{ secrets.HOST }}
        username: ${{ secrets.USERNAME }}
        key: ${{ secrets.SSH_KEY }}
        script: |
          cd /path/to/project
          git pull origin main
          ./update.sh
```

## 总结

本方案采用完全独立部署的架构，具有以下优势：

1. **完全隔离**: 中英文版本完全独立，互不影响
2. **高可用性**: 单个版本故障不影响另一个版本
3. **灵活扩展**: 可针对不同地区进行优化
4. **维护简单**: 容器化部署，便于维护和更新
5. **SEO友好**: 不同域名便于搜索引擎索引

通过本方案，可以为用户提供完整的多语言学习平台体验。
