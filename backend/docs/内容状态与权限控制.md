# 内容状态与权限控制

## 一、内容状态定义

所有用户生成内容 (UGC) 使用统一的 `ContentState` 枚举：

| 值 | 状态 | 含义 |
|---|-----|------|
| 0 | DRAFT | 草稿，用户未提交 |
| 1 | PENDING | 待审核 |
| 2 | PUBLISHED | 已发布，审核通过 |
| 3 | REJECTED | 审核拒绝 |
| 4 | BANNED | 封禁，违规内容 |

## 二、内容关系链

```
课程 (Course) —— 最多两级
├── 子课程 (Course)
└── 节点 (Node)
    └── 帖子 (Post)

角色 (Role)
└── 路线图 (Roadmap)

评论 (Comment) —— 可附加到：
├── 节点 (Node)
├── 帖子 (Post)
├── 路线图 (Roadmap)
└── 评论 (Comment)

记忆卡片组 (MemoryCardDeck)
├── 必须关联：节点 (Node)
└── 可选关联：帖子 (Post)
```

## 三、可见性规则

### 3.1 基本规则

| 内容归属 | 可见状态 |
|---------|---------|
| 自己的内容 | `state != BANNED` |
| 别人的内容 | `state == PUBLISHED` |

### 3.2 入口详情页（检查完整祖先链）

进入详情页时，需要检查内容本身及其完整祖先链的可见性。链上任意一个不可见则返回 403。

| 入口 | 检查链 |
|-----|-------|
| 课程详情 | 课程 → (父课程) |
| 节点详情 | 节点 → 课程 → (父课程) |
| 帖子详情 | 帖子 → 节点 → 课程 → (父课程) |
| 路线图详情 | 路线图 → 角色 |
| 评论上下文 | 评论 → 所属对象的祖先链 |
| 记忆卡片组详情 | 卡片组 → 节点 → 课程 → (父课程)，(帖子) |

### 3.3 列表查询（不检查祖先链）

所有列表只过滤内容自身状态，不检查祖先链：

```sql
WHERE (state = PUBLISHED) 
   OR (creator_id = ? AND state != BANNED)
```

适用于：
- 同一对象下的列表（如帖子下的评论）
- 用户个人空间列表
- 搜索结果列表

**注意**：极少数情况下，用户可能在列表中看到内容，点进去后因祖先不可见返回 403，这是可接受的。

## 四、创建内容限制

创建内容时，祖先链必须全部为 `PUBLISHED` 状态（不区分自己/别人）。

| 创建内容 | 检查 |
|---------|-----|
| 子课程 | 父课程 PUBLISHED |
| 节点 | 课程 → (父课程) 全部 PUBLISHED |
| 帖子 | 节点 → 课程 → (父课程) 全部 PUBLISHED |
| 路线图 | 角色 PUBLISHED |
| 评论 | 被评论对象 → 祖先链 全部 PUBLISHED |
| 记忆卡片组 | 节点 → 课程 → (父课程) 全部 PUBLISHED，(帖子 PUBLISHED) |

## 五、关联内容占位显示

列表或详情中关联的其他内容不可见时，使用占位文案：

| 关联内容 | 占位文案 |
|---------|---------|
| 用户 | "用户已注销" |
| 评论 | "评论已删除" |
| 帖子/节点/课程 | "内容已删除" |

### 5.1 目录类型帖子的特殊处理

帖子 (Post) 有两种类型：
- **文章 (article)**：普通帖子
- **目录 (directory)**：可以引用其他节点

目录帖子的可见性规则：
- 目录本身的可见性：检查目录自身的祖先链（所属节点 → 课程 → 父课程）
- 目录中引用的节点：如果被引用的节点不可见，占位显示 "内容已删除"

## 六、状态变更规则

**不级联更新子内容状态**：
- 父对象被 ban → 子对象状态不变，通过祖先链检查自然不可见
- 父对象恢复 → 子对象自然恢复可见

## 七、搜索索引

| 时机 | 处理 |
|-----|-----|
| 内容 PUBLISHED | 加入索引 |
| 内容被 ban | 从索引删除 |
| 祖先被 ban | 不处理索引（接受短暂不一致） |

## 八、错误码

| 场景 | 错误码 | 错误信息 |
|-----|-------|---------|
| 内容不存在 | NOT_FOUND | 内容不存在 |
| 内容不可见 | CONTENT_NOT_VISIBLE | 内容不可见 |
| 无权查看 | PERMISSION_DENIED | 无权查看此内容 |
| 父内容未发布 | INVALID_PARAMETER | 无法对未发布的内容进行操作 |
