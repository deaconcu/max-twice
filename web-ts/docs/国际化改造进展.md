# Vue前端国际化改造进展文档

## 项目概述

本文档记录Vue前端项目的国际化改造进展，包括已完成的工作、正在进行的任务和待办事项。

## 技术栈

- **Vue 3** + **Vite**
- **vue-i18n@9** - 国际化框架
- **Vuetify** - UI组件库
- **支持语言**: 中文(zh)、英文(en)

## 基础设施 ✅

### 已完成的基础配置

1. **vue-i18n安装和配置**
   - 文件: `src/i18n.js`
   - 支持域名自动检测语言 (`zh.yoursite.com` → 中文, `en.yoursite.com` → 英文)
   - 环境变量覆盖支持 (`VITE_LOCALE`)

2. **语言包结构**
   - `src/locales/zh.json` - 中文语言包
   - `src/locales/en.json` - 英文语言包

3. **构建配置**
   - `vite.config.js` - 支持多语言构建
   - `package.json` - 添加多语言构建脚本

### 构建命令

```bash
# 开发模式
npm run dev:zh    # 中文版开发服务器
npm run dev:en    # 英文版开发服务器

# 构建
npm run build:zh  # 构建中文版到 dist/zh/
npm run build:en  # 构建英文版到 dist/en/
npm run build     # 构建两个版本
```

## 语言包内容 ✅

### 已完成的语言包模块

1. **site** - 网站基本信息
   - name, slogan, about

2. **common** - 通用文本
   - 按钮文本: login, register, save, cancel, etc.
   - 状态文本: loading, success, error, warning

3. **nav** - 导航相关
   - dashboard, learning, courses, roadmap, community, messages, admin

4. **user** - 用户相关
   - login: 登录表单文本
   - register: 注册表单文本
   - profile: 个人资料相关

5. **course** - 课程相关
   - 基本信息: title, description, instructor, duration, difficulty
   - 操作: subscribe, unsubscribe, startLearning, continueLearning
   - 状态: completed, inProgress, notStarted
   - 页面: center, search, createNew, loading, noCourses

6. **sidebar** - 侧边栏
   - 广告: adTitle, adSubtitle, learnMore
   - 操作指南: operationGuide, clickCourse, voteCourse, pinCourse

7. **admin** - 管理后台相关
   - 系统配置: systemConfig, systemOperations
   - 课程管理: courseManagement, professionManagement
   - 内容审核: reviewArticles, reviewComments, approve, reject
   - 操作结果: addSuccess, operationSuccess

8. **learning** - 学习相关
   - 基本信息: title, subtitle, roadmaps, courses
   - 状态管理: all, notStarted, inProgress, completed
   - 操作功能: startLearning, continueLearning, exitLearning
   - 时间格式: timeAgo, duration, estimatedTime

9. **roadmap** - 学习路线图
   - 基本操作: create, edit, delete, save
   - 状态管理: startLearning, voteSuccess, pinSuccess
   - 技能标签: javaBasics, oop, dataStructures, webDevelopment

10. **comment** - 评论系统

- 基本操作: addComment, viewAllComments, reply, upvote
- 状态管理: endOfComments, noComments
- 提示消息: submitSuccess, submitFailed

11. **posting** - 文章/内容相关

- 内容操作: viewFullContent, agree, twiceUnderstand, helpful
- 管理功能: setAsCatalog, cancelSetAsCatalog, pin, unpin
- 反馈消息: operationSuccess

12. **completion** - 课程完成相关

- 庆祝消息: congratulations, courseCompleted, description
- 操作按钮: continueStudy
- 成就标记: courseCompletion, knowledgeMastery

13. **userStats** - 用户统计数据

- 页面标题: title, subtitle, refreshData

- 时间选择: days7, days30, halfYear, oneYear

## 注意事项

### 开发规范

1. **国际化调用方式**

   ```vue
   <!-- 正确 -->
   <template>
     {{ t('user.login.title') }}
     <input :placeholder="t('user.login.email')" />
   </template>

   <!-- 错误 -->
   <template>
     登录
     <input placeholder="邮箱" />
   </template>
   ```

2. **语言包组织**
   - 按功能模块组织 (user, course, nav, etc.)
   - 使用嵌套结构表示层级关系
   - key命名使用小驼峰或点分隔

3. **新增组件**
   - 必须在script中引入 `useI18n`
   - 所有用户可见文本必须使用 `t()` 函数
   - 同步更新中英文语言包

### 维护要求

- 每次添加新的文本内容时，必须同时更新中英文语言包
- 定期检查语言包的完整性和一致性
- 保持语言包的结构清晰，避免深层嵌套
